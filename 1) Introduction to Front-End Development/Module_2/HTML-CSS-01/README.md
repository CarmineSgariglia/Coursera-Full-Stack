# Module 2 – HTML & CSS Practice

This project brings together the foundational knowledge learned in **Module 1** and the first part of **Module 2**, up to the section *“Text and Color in CSS.”*  
Its main goal is to practice core HTML elements and essential CSS rules.

##  Project Structure

### **style.css**
Contains examples of various CSS selectors and tests involving:
- Text properties  
- Colors  
- Basic layout and structure  

### **index.html**
Includes practical exercises using selectors and the main HTML elements covered in the modules, such as:
- Paragraphs  
- Forms  
- Images  
- Headings  
- Lists  
- Links  
- Other fundamental HTML tags  

##  Purpose
The project serves as hands-on practice to reinforce the basics of HTML and CSS, focusing on structure, styling, and selector usage.
